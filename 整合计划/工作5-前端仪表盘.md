│                                                  │
│ 迭代过程：                                       │
│ 1. 查询根服务器 (a.root-servers.net)            │

│ 2. 查询 .com 服务器                             │
│    → 获得 example.com 服务器地址 (38ms)        │
│ 3. 查询 example.com 服务器                      │
│    → 获得最终结果 93.184.216.34 (32ms)         │
│                                                  │
│ 总耗时：115ms (缓存命中：否)                    │
│                                                  │
