## 配置文件示例

### recursive.json（启用）
```json
{
  "enabled": true,
  "port": 5335,
  "root_hints_file": "named.cache",
  "query_timeout": 5000,
  "cache_ttl": 300,
  "max_concurrent_queries": 100
}
```

### recursive.json（禁用）
```json
{
  "enabled": false,
  "port": 5335,
  "root_hints_file": "named.cache",
  "query_timeout": 5000,
  "cache_ttl": 300,
  "max_concurrent_queries": 100
}
```

### config.json（启用本地递归）
```json
{
  "upstreams": [
    {
      "address": "127.0.0.1:5335",
      "type": "local_recursive",
      "priority": 1
    },
    {
      "address": "8.8.8.8:53",
      "type": "external",
      "priority": 2
    }
  ]
}
```
