### 教育模式
```
┌─ 学习模式 ─────────────────────────────────────┐
│                                                  │
│ 输入域名：[example.com] [查询]                  │
│                                                  │
│ 第 1 步：查询根服务器                            │
│ ├─ 发送：example.com? → a.root-servers.net     │
│ ├─ 响应：请查询 .com 服务器 (192.5.6.30)      │
│ └─ 耗时：45ms                                   │
│                                                  │
│ 第 2 步：查询 TLD 服务器                        │
│ ├─ 发送：example.com? → 192.5.6.30 (.com)     │
│ ├─ 响应：请查询 example.com 服务器 (93.184...)│
│ └─ 耗时：38ms                                   │
│                                                  │
│ 第 3 步：查询权威服务器                         │
│ ├─ 发送：example.com? → 93.184.216.1           │
│ ├─ 响应：example.com A 93.184.216.34           │
│ └─ 耗时：32ms                                   │
│                                                  │
│ 总耗时：115ms                                   │
│ 缓存状态：新查询（无缓存）                       │
│ 隐私状态：✅ 完全隐私（查询未离开本地）         │
│                                                  │
└──────────────────────────────────────────────────┘
```
